# Chart Understanding Agent

A production-grade multimodal agent for analyzing charts and graphs using Groq's Vision API. This tool can detect chart types, extract structured data, and answer questions about visual data.

## Features

- **Chart Type Detection**: Automatically identifies bar charts, line charts, pie charts, scatter plots, and more
- **Data Extraction**: Converts visual chart data into structured JSON and CSV formats
- **Question Answering**: Answer analytical questions about charts
- **Data Validation**: Built-in validation and consistency checking
- **CLI Interface**: Easy-to-use command-line interface with rich formatting
- **Professional Logging**: Comprehensive logging with Loguru

## Supported Chart Types

- Bar Charts (horizontal and vertical)
- Line Charts (single and multi-series)
- Pie Charts
- Scatter Plots
- Area Charts
- Combo Charts
